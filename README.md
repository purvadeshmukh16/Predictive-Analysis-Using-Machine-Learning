# Predictive-Analysis-Using-Machine-Learning
Predictive analysis project using Machine Learning with Titanic dataset. Includes feature selection, model training, and evaluation.

## Steps Covered
- Data loading and understanding
- Feature selection and data cleaning
- Train-test split
- Model training using Logistic Regression
- Model evaluation using accuracy, classification report, and confusion matrix

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Dataset
Titanic dataset (publicly available)
